Installing the Polls module

Installing a Zikula module is a simple process

    * Copy contents of the downloaded archive to your modules directory.
    * Login to your site as an admin.
    * Go to the Extensions administration page.
    * Locate the entry for the new module and install it.

At this point your new module is installed and you can begin to create new polls.
